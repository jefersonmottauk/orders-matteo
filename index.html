🗄️ Monthly Security Backup

It's a new month — have you exported a backup copy of your data from Supabase this month?

In the Supabase SQL Editor, run this once, then click Export → CSV on the result (a single file with every table):
select 'orders_history' as table_name, to_jsonb(t) as row_data from orders_history t union all select 'users', to_jsonb(t) from users t union all select 'products', to_jsonb(t) from products t union all select 'admin_messages', to_jsonb(t) from admin_messages t union all select 'app_settings', to_jsonb(t) from app_settings t;
Also go to Storage → product-images, select all files and download them (product photos aren't included in the query above).
